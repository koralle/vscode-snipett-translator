# vscode-snipett-translator

## 概要

* VSCodeのスニペットを書くのがしんどい、特に競プロでペタペタする長いやつ
* VSCodeで書いたソースをVSCodeのスニペットに変換したい
* 既存ツールはある、むしろ上位互換だけどあくまで勉強用に作る

## 使う技術

* TypeScript
* Redux
* React
* Webpack
* Git/GitHub
* Sass/Scss
* Styled Component
* Containers/Presentation Component
* Material UI
* React-Testing-Library
* なんちゃってCI/CD（できたら）
